# Hand Tracker — Setup

## Requisiti

- Python 3.11

---

## Installazione

### 1. Estrai lo zip e entra nella cartella

```bash
unzip hand-tracker-backup.zip
cd hand-tracker
```

### 2. Crea il virtual environment

```bash
python3.11 -m venv venv
```

### 3. Attiva il venv

**macOS / Linux:**
```bash
source venv/bin/activate
```

**Windows:**
```bash
venv\Scripts\activate
```
